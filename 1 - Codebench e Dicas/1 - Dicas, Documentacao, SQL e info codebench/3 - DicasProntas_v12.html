SyntaxError: invalid syntax

Parece que há um erro na linha indicada ou anterior.
Um erro de sintaxe ocorre quando alguma regra da linguagem de programação não é seguida. Vamos lembrar das mais comuns?


1 - Toda string (sequência de caracteres) deve ser definida entre um par de aspas. Podem ser aspas simples (') ou duplas ("), mas nunca misture as duas:

Exemplo com erro Exemplo corrigido 
print( Bruce Wayne") 

print( Barry Allen ) 
                               
x =  Viuva Negra 
              

print("Bruce Wayne") 

print('Barry Allen') 

x = "Viuva Negra"

2 - No Python 3.x, toda função deve indicar seus argumentos entre um par de parênteses, e os argumentos devem ser separados por vírgulas:

Exemplo com erro Exemplo corrigido 
print "Hermione Granger" 
                        
round(var, 3 
           

print("Hermione Granger")

round(var, 3)

3 - Comandos de condição (if, else, elif) e repetição (for, while) devem terminar com sinal de dois pontos:

Exemplo com erro Exemplo corrigido 
# SEM dois pontos
if (x == 1)     
    print(x)

while (i < 10) 
             
    
# COM dois pontos
if (x == 1):
    print(x)

while (i < 10):

NameError: name 'NOMEDAVARIAVEL' is not defined

Parece que há um erro na linha indicada ou anterior.
Ao usar uma variável ou função, verifique se elas foram definidas. Se sim, verifique se elas foram usadas com o mesmo nome com que foram definidas. Observe o uso de letras maiúsculas e minúsculas.

Exemplo com erro Exemplo corrigido 
Area = 10 
print(7 + area) 

area = 10 
print(7 + area)

ValueError: invalid literal for int() with base 10

Parece que há um erro na linha indicada ou anterior.
A função int() devolve um valor inteiro como resultado.
  • Caso precise de um valor float (número decimal), utilize o formato x = float(input()):
    • Exemplo com erro Exemplo corrigido 
     x = int(input("Digite o preco: ")) 

     x = float(input("Digite o preco: "))

  • Caso precise de uma string (sequência de caracteres), utilize o formato x = input():
    • Exemplo com erro Exemplo corrigido 
     x = int(input("Digite o nome: ")) 

     x = input("Digite o nome: ")

  • Caso precise ler mais de um valor inteiro, utilize um comando input() para cada entrada:
    • Exemplo com erro Exemplo corrigido 
     x = int(input("Digite as idades das pessoas: ")) 

     x1 = int(input("Digite a idade da pessoa 1: "))
 x2 = int(input("Digite a idade da pessoa 2: "))

EOFError: EOF when reading a line

Parece que há um erro na linha indicada ou anterior.
O código está solicitando mais entradas (inputs) do que o exigido pelo enunciado do exercício. Corrija o número de funções input() no código.

IndentationError: expected an indented block

Parece que há um erro na linha indicada ou anterior.
  • Qualquer estrutura do Python que termine com dois pontos (if, elif, else, while, for) deve ter seu conteúdo identado para a direita a fim de ser considerado como parte dela.
  • Certifique-se que a linha onde o erro foi encontrado está com a mesma indentação da linha de cima.
Exemplo com erro Exemplo corrigido 
x = input()
if (x == "1"):
    print("resultado = 1")
if (x == "2"):
print("resultado = 2")

x = input()
if (x == "1"):
    print("resultado = 1")
if (x == "2"):
    print("resultado = 2")

IndentationError: unindent does not match any outer indentation level

Parece que há um erro na linha indicada ou anterior.
  • Elimine espaços digitados à frente dos comandos if, elif e else.
  • Se duas condições estão no mesmo nível lógico de comparação, então aplique a mesma indentação nos comandos if, elif e else correspondentes.
  • Para indentar o código, use somente a tecla TAB. Não misture TAB e espaços.
Exemplo com erro Exemplo corrigido 
a = 1
if (a == 1):
    print("1")
 if (a == 2):
    print("2")

a = 1
if (a == 1):
    print("1")
if (a == 2):
    print("2")

IndexError: index X is out of bounds for axis Y with size Z


Parece que há um erro na linha X.
  • O código está tentando acessar uma linha ou coluna fora dos limites do vetor ou da matriz.
  • Verifique se o número de interações do laço excede a quantidade de elementos do vetor ou matriz.
  • Lembre-se: os índices são numerados a partir de 0 (zero).

Exemplo com erro Exemplo corrigido 
from numpy import *

# vetor de 4 elementos:
vet = array([0,1,2,3])

# quinto elemento:
print(vet[4])

from numpy import *

# vetor de 4 elementos:
vet = array([0,1,2,3])

# quarto elemento:
print(vet[3])

TypeError: unsupported operand type(s)


Parece que há um erro na linha indicada ou anterior.
  • Verifique se você confundiu o uso de alguns símbolos nas operações. Por exemplo, use ** para representar a exponenciação.
  • Verifique se você usou ponto (.) em vez da vígula (,) para separar a parte inteira da parte decimal de um número real.
Exemplo com erro Exemplo corrigido 
# Sinal de expoente trocado
a = 2 ^ 0.5

# Uso equivocado de vírgula para separar casas decimais
round(2**(0,5), 8)

# Uso correto de 2 asteriscos
a = 2 ** 0.5

# Uso correto do ponto em números reais
round(2**(0.5), 8)
