Criação de "dicas" para o Codebench


As dicas apresentadas aqui não são definitivas e podem ser alteradas diante das respostas dos "especialista" (obtidas via formulário), discussão do grupo de pesquisa e análise de nova literatura científica relacionada*


Para gerar a estrutura das dicas, estão sendo analisadas as literaturas científicas relacionadas a:


1 - Geração de feedback relevante, seguindo guidelines baseadas no estudo de carga cognitiva e áreas relacionadas da psicologia;
2 - Outros estudos de geração de feedback, em computação (nebulosa e sem consenso);
3 - Estudos de caso de melhoramento de mensagens de erro de compiladores (Java, C++);

Para gerar os exemplos de dicas, estão sendo analisados:


1 - Exemplos de submissão dos alunos, analisados via banco de dados do Codebench;
2 - Documentação do Python relacionado aos erros;
3 - Material do Stack Overflow relacionado aos erros.
4 - Listagem de possíveis causas para o erro (seção "possíveis causas").

NameError:

#Codigo do aluno
print(8*7 + x)

#Possíveis causas (não faz parte da dica)
1 - Você pode estar tentando usar uma variável que não foi declarada EXATAMENTE com o mesmo nome, cheque se o nome da variavel que voce esta tentando usar tem o mesmo nome da variavel declarada;
2 - Você pode estar tentando imprimir uma string sem usar a função "print" do python;
3 - Você pode estar errando o nome de uma função padrão do python.

#Erro original do interpretador 
        $ python3 main.py
        Traceback (most recent call last):
          File "main.py", line 1, in 
            print(8*7 + x)
        NameError: name 'x' is not defined
#Dica "Placebo"
ERRO: O nome 'x' não está definido!

#Versões da dica:
#v1.0 "NameError" geralmente indica a tentativa de uso de uma variável que não está definida
Veja se está utilizando o nome da variável corretamente, ou se a variável que está sendo chamada foi criada

#v1.1
"NameError" geralmente indica a tentativa de uso de uma variável que não está definida
Veja se o nome da variável está sendo utilizado corretamente, ou se a variável que está sendo chamada foi criada

#v2.0
Parece que você tem um erro na linha 4
Veja se a variável que você está tentando usar foi criada e tem o mesmo nome

#v2.1
Parece que você tem um erro na linha 4
Veja se a variável que está sendo chamada foi criada e tem o mesmo nome

#v2.2
Parece que existe um erro na linha 4
Veja se a variável que está sendo chamada foi criada e tem o mesmo nome

#v2.3
Parece que existe um erro na linha 4
Veja se a variável ou função que está sendo chamada existe e tem o mesmo nome

#Exemplo do tipo 2 de dica:

Código com erro: 
X = 10 

print(7 + x)
Código corrigido: 
X = 10 

print(7 + X)

Parece que existe um erro na linha 4
Veja se a variável ou função que está sendo chamada existe e tem o mesmo nome

IndentationError: expected an indented block:

#Codigo do aluno 
x = input()
if x == "1":
    print("resultado = 1")
if x == "2":
print("resultado = 2")
#Possíveis causas (não faz parte da dica)
1 - O código do erro está alinhado ou mais à esquerda do bloco acima.

#Erro original do interpretador 
        $ python3 main.py
        File "main.py", line 5
            print("resultado = 2")
                ^
        IndentationError: expected an indented block
#Dica "Placebo"
ERRO: O bloco deve estar indentado!

#Versões da dica:
#v1.0
Parece que existe um erro na linha 5

#v1.1
Parece que existe um erro na linha 5
Veja se a código do erro está mais à esquerda ou bem abaixo da linha acima

#v1.1
Parece que existe um erro na linha 5
